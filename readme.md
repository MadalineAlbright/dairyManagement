
# Stock-Management
This is a demo of stock management system.

# Techstack
  - Laravel 5.5
  - JS
  - Laravel Mix
  - Bootstrap
  - webpack
  - node Modules
# Features
  - Stock Adding
  - Stock Manage / Edit / Add stock to old invoice
  - PDF Generate
  - Supplier Management
  - Supplier Payment Module
  - Settings for brand and style/Product Category Manage
  - Payment method Module
